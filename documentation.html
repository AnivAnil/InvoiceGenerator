📚 Documentation

Welcome to POS Retail Pro! This guide will help you get started with the platform quickly.

Getting Started

  • Create an Account: Click Get Started on the home page and fill in your details. An admin will approve your account.
  • Login: Once approved, use your username/email and password on the Sign In page.
  • Dashboard: After login you'll see your invoice ledger, stats, and admin panel (if you're an admin).

Creating an Invoice

  • Click + New Invoice on the dashboard.
  • Fill in your store details (auto‑saved for future use).
  • Add the customer's information under Bill To.
  • Set CGST, SGST, Cess percentages manually for each invoice.
  • Add products/services using the + Add Item button. Each row calculates automatically.
  • Use Save Invoice to store it, or Print / PDF to generate a professional document.

Invoice Ledger & Filtering

On the dashboard, you can filter invoices by month and year. The total revenue for the selected period updates instantly.

Admin Approval

If you're the admin (ANIV6536 by default), you'll see an Admin Panel on the dashboard. From there you can:

  • Approve or reject pending user registrations.
  • View a list of all approved users.

Cross‑Device Sync

Use the Export button (on dashboard or create invoice page) to download your full account data. On a new device, open the login page and click Restore from Backup to import everything – account, invoices, shop details, logo.

Keyboard Shortcuts

  • Ctrl+S (or Cmd+S on Mac) – Save invoice.
  • Ctrl+P – Print invoice.
